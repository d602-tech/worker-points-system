# 115年度協助員點數管理系統 — 身分別權限對照表

> 版本：v3.2.7｜更新時間：2026-05-04  
> 本文件依程式碼實際實作為準，描述各身分在系統中可執行的所有操作。

---

## 系統身分架構

系統共有 **2 大存取介面、4 種登入角色、4 種協助員職務類型**。

```
登入角色（role）
├── worker          → 進入【協助員端】行動介面
├── deptMgr         → 進入【管理端】桌面介面
├── billing         → 進入【管理端】桌面介面
└── admin           → 進入【管理端】桌面介面（最高權限）

協助員職務類型（workerType，僅 worker 角色適用）
├── general         → 一般工地協助員（大潭/通霄/興達/大林）
├── offshore        → 離島工地協助員（金門/琉球）
├── safety          → 職安業務兼管理員（處本部）
└── environment     → 環保業務人員（處本部）
```

## 一、協助員端（worker）

> 登入後自動導向 `/worker/today`，僅能使用行動端介面。

### 1-A. 每日填報（TodayTasks）
| 功能 | 說明 |
|------|------|
| ✅ 瀏覽今日/歷史填報 | 可向前/向後切換日期，查看每日填報狀態 |
| ✅ 勾選完成項目 | 僅可勾選「草稿 draft」或「已退回 rejected」狀態的項目 |
| ✅ 上傳佐證附件 | 拍照、選相簿、選文件（.jpg/.png/.pdf），每個項目可多份 |
| ✅ 填寫作業備註 | 附加文字說明至各項目 |
| ✅ 送出今日填報 | 勾選項目並上傳佐證後，按「確認送出」進入稽核流程 |
| ✅ 重新上傳佐證 | 已送出項目被退回後，可填寫修改原因並重新覆寫佐證 |
| ❌ 修改已送出資料 | 已送出（submitted/approved）無法自行編輯，需管理員退回後重填 |
| ❌ 查看其他人員資料 | 只能看自己的填報紀錄 |

**點數項目由 workerType 決定**（依 `POINTS_CONFIG_SEED` 中 category=A1 的項目）：
- `general` / `offshore`：一般工地每日作業項目
- `safety`：職安業務每日作業項目
- `environment`：環保業務每日作業項目

### 1-B. 日曆總覽（CalendarOverview）
| 功能 | 說明 |
|------|------|
| ✅ 月曆視圖 | 查看整月出勤狀態（已通過/已送出/草稿/退回/假日/請假） |
| ✅ 列表視圖（出勤計畫表） | 查看並直接修改當月＋下個月每日上午/下午出勤狀態 |
| ✅ 申請請假 | 選擇請假日期、時間區間、假別（特休/病假/事假/婚假/喪假/公假），上傳假單佐證 |
| ✅ 點擊日期開啟 Drawer | 查看該日已填報的任務、佐證檔案、點數 |
| ✅ 本月預估點數 | 依當前出勤狀態自動估算本月可能總點數 |
| ✅ 未完成提醒 | 顯示過去工作日中尚未填報出勤或未上傳佐證的日期清單 |
| ✅ 月報待填項目提醒 | 在月曆頁顯示尚未送出的 B1/B2 例行月報項目 |
| ❌ 修改已鎖定出勤 | `isFinalized=true` 的差勤記錄不可更動 |

### 1-C. 月報填報（MonthlyReport）
| 功能 | 說明 |
|------|------|
| ✅ 查看例行月報項目 | 顯示依 workerType 對應的 B1（例行）、B2（農曆春節加給）項目 |
| ✅ 勾選完成 B1/B2 項目 | 勾選即表示本月已完成，可附上佐證後送出 |
| ✅ 上傳 B1/B2 佐證 | 拍照或選擇檔案，B1/B2 項目**必須上傳佐證才能送出** |
| ✅ 查看 C 類績效欄位 | 可看到績效評級（優/佳/平）但**無法自行選擇** |
| ✅ 送出月報 | 送出全部已完成且有佐證的項目 |
| ✅ 重新覆寫已送出佐證 | 已送出項目可填修改原因後重新上傳佐證 |
| ✅ 查看本月累計點數 | 顯示每日點數 + 例行月報 + 績效評估的加總 |
| ❌ 自行填寫 C 類績效 | C 類績效（優/佳/平）**由管理端評估，協助員無法自行設定** |

### 1-D. 差勤排班（AttendanceSchedule）
| 功能 | 說明 |
|------|------|
| ✅ 預排下個月班表 | 系統預設依行事曆計算標準工作日，可自行調整 |
| ✅ 設定請假日 | 點選日期設定請假時數、假別、時間區間（自動計算扣除午休） |
| ✅ 申請調班 | 選擇上班日與假日互換，需填寫原因 |
| ✅ 儲存排班 | 送出有異動的日期，僅送出有變更的天數 |

| ❌ 排定超出標準工時 | 系統自動驗證不可超過標準上班天數 |

## 二、管理端（admin / deptMgr / billing）

> 登入後自動導向 `/admin/users`，使用桌面版介面。

### 共同說明
- **所有管理端角色**均可進入管理端介面（`role !== "worker"` 即放行）。
- **更新歷程（Changelog）頁面**僅 `admin` 可見（頁籤過濾：`t.id !== "changelog" || user?.role === "admin"`）。

---

### 2-A. 人員管理（AdminUsers）— 所有管理端角色
| 功能 | 說明 |
|------|------|
| ✅ 查看所有協助員清單 | 含工號、姓名、部門、區域、職務類型、到職日、年資天數、狀態 |
| ✅ 搜尋人員 | 依姓名、工號、Email 關鍵字搜尋 |
| ✅ 篩選人員 | 依協助員類型或服務區域篩選 |
| ✅ 新增人員 | 填寫基本資料、服務區域（自動對應職務類型）、過往年資區間並上傳佐證 |
| ✅ 編輯人員 | 修改既有人員資料（姓名/Email/部門/區域/職務類型/到職日/在職狀態/年資） |
| ✅ 停用人員 | 將在職狀態設為「離職」或「停職」 |
| ✅ 設定登入密碼 | 為人員設定密碼登入憑證（SHA-256 雜湊後存至 Google Sheets） |
| ✅ 匯出人員資料 | 匯出功能（需 API 串接，目前顯示提示） |

**服務區域 ↔ 職務類型對應規則**：
| 服務區域 | 職務類型 |
|---------|---------|
| 大潭 / 通霄 / 興達 / 大林 | 一般工地協助員（general） |
| 金門 / 琉球 | 離島工地協助員（offshore） |
| 處本部 | 職安業務管理員（safety）或環保業務人員（environment）— 可選擇 |

### 2-B. 差勤管理（AdminAttendance）— 所有管理端角色
| 功能 | 說明 |
|------|------|
| ✅ 查看全部協助員出勤資料 | 依月份、協助員姓名/工號篩選出勤記錄 |
| ✅ 確認/鎖定出勤 | 將差勤記錄設為已確認（isFinalized），鎖定後協助員無法再編輯 |
| ✅ 查看請假申請 | 含佐證附件預覽連結 |
| ✅ 匯出出勤報表 | 匯出 Excel 格式出勤彙整 |

---

### 2-C. 審核中心（ReviewCenter）— 所有管理端角色
| 功能 | 說明 |
|------|------|
| ✅ 查看待審核項目 | 顯示各協助員送出的月報/日報填報項目 |
| ✅ 依狀態篩選 | 全部 / 待初審 / 初審通過 / 廠商確認 / 已請款 / 已退回 |
| ✅ 初審通過 | 對「待初審（submitted）」的項目執行初審通過 → 狀態變更為 `dept_approved` |
| ✅ 退回修改 | 在任一審核階段退回，需填寫退回原因（協助員可看到） |
| ✅ 廠商確認 | 對「初審通過」的項目確認廠商核准 → 狀態變更為 `billing_confirmed` |
| ✅ 廠商退回 | 廠商核准階段退回，需填寫原因 |
| ✅ 已請款 | 對「廠商確認」的項目標記為已請款 → 狀態變更為 `billed` |
| ✅ 查看佐證附件 | 可點擊查看已上傳的佐證文件連結 |

**審核流程**：
```
草稿 → 已提交（submitted）→ 初審通過（dept_approved）→ 廠商確認（billing_confirmed）→ 已請款（billed）
                ↕ 任一階段均可退回修改（rejected）
```

## 四、協助員職務類型對應說明

> 此分類決定協助員在填報系統中可看到的**點數項目清單**，與登入角色（worker）分開設定。

| workerType | 中文名稱 | 所在區域 | 說明 |
|-----------|---------|---------|------|
| `general` | 一般工地協助員 | 大潭、通霄、興達、大林 | 標準工地環安協助作業項目 |
| `offshore` | 離島工地協助員 | 金門、琉球 | 離島加給區域，項目與 general 相同 |
| `safety` | 職安業務兼管理員 | 處本部 | 職安相關行政業務，不同點數項目清單 |
| `environment` | 環保業務人員 | 處本部 | 環保相關行政業務，不同點數項目清單 |

## 五、登入方式

| 登入方式 | 說明 |
|---------|------|
| **Google 帳號登入** | 透過 Google Identity Services，系統以 email 查詢 GAS 人員資料 |
| **帳號密碼登入** | Email + 密碼（SHA-256 雜湊後比對 Google Sheets「密碼雜湊」欄），適合無 Google 帳號者 |

> ⚠️ **注意**：帳號必須已由管理端建立至人員資料表，否則登入驗證雖通過仍無法取得資料（回傳「找不到此帳號」）。
